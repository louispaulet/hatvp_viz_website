# Project Instructions

- Always read this `AGENTS.md` before starting work in this repository.
- This project is a Vite + React + Tailwind CSS dashboard for HATVP declaration visualizations.
- Prefer React components and shared data utilities over ad hoc DOM manipulation.
- Keep the interface in French unless the user asks otherwise.
- Preserve the GitHub Pages custom domain: `hatvp.thefrenchartist.dev`.
- Avoid reverting user changes. Check `git status` before large edits.
- Validate meaningful changes with `make test` and `make build`.
- After every completed change, commit and push the work unless the user explicitly asks not to.
